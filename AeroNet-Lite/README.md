# AeroNet-Lite

Autonomous drone delivery simulation with a data science and AI pipeline.

The actual project files are stored under `project/`.

## Structure

- `project/data/` — raw and processed datasets
- `project/src/` — Python source code and simulation modules
- `project/notebooks/` — Jupyter notebooks for analysis
- `project/report/` — project report and figures
- `project/README.md` — full project instructions and run guide

## Notes

- Open `project/README.md` for detailed setup and execution steps.
- The top-level folder is intentionally a container for the main `project/` workspace.
